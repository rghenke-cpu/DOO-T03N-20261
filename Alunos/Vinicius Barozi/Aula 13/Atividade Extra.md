### O que é?

Generics permitem criar classes e métodos que trabalham com diferentes tipos de dados de forma segura, evitando conversões desnecessárias.

### Para que serve?

Servem para reutilizar código e aumentar a segurança de tipos durante a compilação.

### Como é normalmente utilizado?

É muito utilizado em coleções do Java, como List, ArrayList e Map.

### Exemplo

```java
List<String> nomes = new ArrayList<>();

nomes.add("João");
nomes.add("Maria");
